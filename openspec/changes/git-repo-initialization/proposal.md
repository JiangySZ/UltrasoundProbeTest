## Why

项目目前缺乏版本控制，代码无法追踪历史、协作管理，也无法配置CI/CD流水线。初始化Git仓库是项目开发的基础设施需求。

## What Changes

- 初始化本地Git仓库
- 配置.gitignore忽略编译产物和临时文件
- 创建初始项目目录结构
- 首次提交项目基础结构
- 配置Git远程仓库（可选）

## Capabilities

### New Capabilities
- `git-initialization`: Git仓库初始化与基础配置

### Modified Capabilities
（无）

## Impact

- 新增.gitignore配置
- 新增README.md项目说明
- 项目代码纳入版本控制
